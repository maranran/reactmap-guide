iconConfig

icon参数详情

名称 类型 说明
size Array[width, height] 图标尺寸，默认值[36,36]
image String 图标的取图地址
imageSize Array[width,height] 图标所用图片大小，根据设置的大小拉伸或者压缩图片，等同于css中的background-size属性
imageOffset Array[x,y] 图标取图偏移量，相对于图片左上角的相对像素坐标，默认为[0，0]
